# Parus RX

Parus RX is a group of microservices that provide an integration layer between the Parus and the other solutions and services.

Parus 8 is a Russian ERP (Enterprise Resource Planning) software developed by the corporation Parus. It is designed to help businesses manage their financial, accounting, and operational processes. The software includes modules for accounting, payroll, inventory management, sales, purchasing, and more. It is widely used in Russia and other CIS countries.
